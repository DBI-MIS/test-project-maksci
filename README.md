# 🚀 Technical Intern
### Welcome! This repository serves as a technical training project for the internship program. The intern/s is/are tasked with building a localized module using the **LIVT Stack**.

---

## 🛠 Tech Stack
* **Backend:** Laravel 12 (PHP 8.4+)
* **Frontend:** Vue 3
* **Bridge:** Inertia.js with Wayfinder
* **Language:** TypeScript (Strict Mode)
* **Styling:** Tailwind CSS
